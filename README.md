# IoT-Driven Piezoelectric Flooring System for Smart Energy Conversion

## 📘 Project Overview
The IoT-Driven Piezoelectric Flooring System is a smart energy generation project designed to convert mechanical pressure from human footsteps into electrical energy using piezoelectric sensors.

The generated energy is monitored and visualized using IoT technologies such as ESP8266 and the Blynk platform. This project demonstrates renewable energy generation and smart monitoring concepts.

---

## 🎯 Objectives
- Generate electricity from mechanical pressure
- Demonstrate renewable energy harvesting
- Monitor generated energy using IoT
- Visualize energy data remotely

## 🚀 Working Principle
1. Piezoelectric sensors generate voltage when pressure is applied.
2. Generated electrical signals are collected and processed.
3. Arduino reads and monitors the generated energy.
4. ESP8266 sends the data to the Blynk platform.
5. Energy data is visualized remotely through the mobile application.
